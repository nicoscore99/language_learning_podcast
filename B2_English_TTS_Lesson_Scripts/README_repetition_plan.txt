Production TTS Lesson Package - Repetition Plan

Teacher language: Mandarin Chinese, tagged as <lang code="zh" />.
Target language: English, tagged as <lang code="en" />.
English variety: Neutral international English.
TTS format: Plain .txt files for a downstream Python TTS generator. Each spoken block is preceded by an explicit language tag. The only tags used in lesson files are <lang code="..." /> and <break time="...s" />.
Pause rule: Pauses are variable and based on task difficulty. Short answer-reflection pauses after model answers have been lengthened to about 1.5 seconds, so learners have more time to absorb and repeat the correct answer. Recall and translation pauses are generally longer than in the previous package, and no individual break exceeds 10.0 seconds.
Repetition strategy: Each lesson introduces six target words, asks for immediate repetition, recalls the words in mixed order, combines current words with earlier lesson words, includes mini-dialogue practice, uses practical question-answer prompts, and ends with long-answer practice plus final review. From lesson 2 onward, earlier vocabulary is recycled cumulatively, with priority given to recent lessons and occasional older review.
