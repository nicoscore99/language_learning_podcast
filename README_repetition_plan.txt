Teacher language: Mandarin Chinese.

Target/student response language: English.

English variety: Neutral international English.

TTS format: Plain .txt files designed for direct use in ElevenLabs Multilingual v2 or a similar TTS tool. Lesson files contain only spoken Mandarin prompts, spoken English model answers, and pause tags in the exact format <break time="3.0s" />.

Pause rule: Every break tag is exactly <break time="3.0s" />. Longer thinking time is created only by stacking several 3-second break tags. No single break exceeds 3.0 seconds.

Repetition strategy: Each lesson introduces six target words, gives Mandarin meaning prompts, provides B1-level English model sentences, and uses recall in mixed order. From lesson 2 onward, lessons include cumulative review of recent and earlier vocabulary. Practice moves from word recall to sentence translation, mixed sentence building, mini-dialogues, practical B1 question-answer work, long-answer paragraph practice, and final review. The scripts use guided repetition without speaker labels or production notes so the audio remains clean and natural.
