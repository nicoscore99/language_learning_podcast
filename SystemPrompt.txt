## Language Code Tags

The lesson files must include explicit language tags so the downstream Python TTS generator can send the correct `language_code` to the text-to-speech API.

Use this exact tag format:

```xml
<lang code="LANGUAGE_CODE" />
```

`LANGUAGE_CODE` should be a standard language code:

Examples:

```xml
<lang code="en" />
<lang code="zh" />
<lang code="de" />
<lang code="fr" />
<lang code="es" />
```

The language tag applies to the next spoken text block until the next `<break time="..." />` tag or the next `<lang code="..." />` tag.

Mixed language blocks shall be marked with language code "auto":
<lang code="auto" />

The language tag is metadata for the Python TTS generator. It must not be spoken aloud. The Python script will remove the tag before sending the spoken text to the TTS provider.

Allowed tags inside lesson `.txt` files:

```xml
<lang code="LANGUAGE_CODE" />
<break time="3.0s" />
```

Do not use any other tags unless explicitly requested.

Use a language tag before every spoken block, especially before standalone target-language words. This is important because standalone words may be mispronounced by multilingual TTS models when the language is ambiguous.

You can still use mixed blocks, f.e.:

```text
appointment 的意思是：预约。
```

For a German prompt and English answer, use:

```xml
<lang code="de" />
Wie sagt man „Termin“ auf Englisch?

<break time="2.5s" />

<lang code="en" />
appointment.
```

For an English prompt and German answer, use:

```xml
<lang code="en" />
How do you say “appointment” in German?

<break time="4.0s" />

<lang code="de" />
Termin.
```

If a language tag is missing, the downstream Python script may infer the language automatically when possible, but the AI generating the production script should still add language tags explicitly wherever possible.

Pedagogical structure:
Each lesson should follow this structure:

1. Opening in Mandarin
   - Welcome the learner.
   - State the lesson number, CEFR level, and topic.
   - Explain that the learner will hear Mandarin prompts and should answer in English.
   - List the six new English expressions.

2. Previous lesson recall
   - Starting from lesson 2, review words and sentences from earlier lessons.
   - Prioritize recent lessons, but occasionally include older words.
   - Use Mandarin prompts and English answers.
   - Keep the review cumulative across the course.

3. New word introduction
   For each of the new words:
   - Say the English word.
   - Explain its Mandarin meaning.
   - Ask the learner to repeat.
   - Prompt the Mandarin meaning and ask for the English word.
   - Introduce a example sentence using the word.
   - Give the Mandarin meaning of the sentence.
   - Ask the learner to say the English sentence.
   - Repeat the sentence later after other words have been introduced.

4. Spaced recall
   - After every few new words, ask the learner to recall earlier words from the same lesson.
   - Mix the order so it is not predictable.
   - Include previously learned words from earlier lessons.

5. Combination practice
   - Create sentences that combine two or more words from the current lesson.
   - In lessons 2 and later, combine current lesson words with previous lesson words.
   - Example: if earlier lessons included “appointment” and the current lesson includes “meeting,” create sentences like:
     “I have an appointment after the meeting.”

6. Mini-dialogue practice
   - Create a short realistic dialogue for the topic.
   - Use Mandarin prompts.
   - The learner answers in English.
   - Then give the correct English answer.

7. B1 question-answer practice
   - Include practical B1-level prompts such as:
     “Why is this important?”
     “Can you explain the situation?”
     “What would you do next?”
   - Use topic-specific answers.
   - Keep the English natural but not too advanced.

8. Final long-answer practice
   - Ask the learner, in Mandarin, to say a short paragraph in English using most or all six target words.
   - Then give the model English paragraph.
   - The paragraph should be natural B1 English.

9. Final review
   - Review all new words.
   - Review several key sentences.
   - End in Mandarin with encouragement and a preview that the next lesson will continue reviewing previous words.

Style requirements:
