Teacher language: Mandarin Chinese, tagged primarily with <lang code="zh" />.
Target/student response language: English, tagged with <lang code="en" />.
English variety: neutral international English.
TTS format: plain .txt files for processing by a Python TTS generator. Lesson files contain only language tags, spoken Mandarin prompts, spoken English model answers, and break tags.
Pause rule: pauses are adaptive and have been made generally longer than the previous adaptive-break version. Short word recall generally uses about 3.5 to 4.8 seconds, sentence translation and combination practice use longer pauses, practical question-answer prompts use longer thinking time, and final long-answer practice uses up to 9.8 seconds. No individual break exceeds 10.0 seconds. Reflection pauses after English model answers are deliberately longer, generally about 2.0 to 3.0 seconds, so learners have time to process and repeat the correct answer.
Repetition strategy: each lesson introduces six target words, practices immediate repetition, Mandarin-to-English recall, example sentences, delayed recall, mixed review, combination sentences, mini-dialogues, B1 practical answers, a final paragraph, and a final review. Lessons 2 onward include cumulative recall from earlier lessons, prioritizing recent vocabulary while occasionally revisiting older words.
Language tagging strategy: <lang code="zh" /> appears before Mandarin teacher prompts and <lang code="en" /> appears before English words, model answers, and target-language sentence blocks. Mixed blocks are allowed by the prompt but are avoided where practical for pronunciation reliability.
